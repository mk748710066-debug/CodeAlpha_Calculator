# 🧮 Basic Arithmetic Calculator in C

## 📌 Project Description

This project is a simple **Arithmetic Calculator** developed using the **C Programming Language**.
The program performs basic mathematical operations using a **switch case** statement.

It allows the user to enter two numbers and select an operator to perform the desired calculation.

---

## 🚀 Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Division by zero error handling
* User-friendly input/output

---

## 🛠️ Technologies Used

* C Programming Language
* GCC Compiler / Turbo C
* Command Line Interface (CLI)

---

## 📂 Program Logic

1. User enters the first number.
2. User selects an operator (`+`, `-`, `*`, `/`).
3. User enters the second number.
4. Program uses **switch-case** to perform the operation.
5. Result is displayed on the screen.

---

## 💻 How to Run the Program

### Step 1: Compile

```bash
gcc calculator.c -o calculator
```

### Step 2: Execute

```bash
./calculator
```

---

## 🧾 Example Output

```
Enter first number: 10
Enter an operator (+, -, *, /): +
Enter second number: 5
Result = 15.00
```
